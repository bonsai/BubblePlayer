# 🫧 光るシャボン玉レコードプレーヤー

Three.js × React で作る幻想的なレコードプレーヤー

## 特徴

- 🎵 回転するレコード盤
- 🫧 虹色に光るシャボン玉
- ✨ 透過・屈折エフェクト
- 🎨 リアルタイムアニメーション
- 🎮 Play/Pause コントロール

## セットアップ

```bash
cd GAME/bubble-record-player
npm install
npm run dev
```

## 技術スタック

- React 18
- Three.js
- @react-three/fiber
- @react-three/drei
- Vite
- TypeScript

## 実装内容

- レコード盤の回転アニメーション
- シャボン玉の浮遊・発光エフェクト
- MeshTransmissionMaterial による透過表現
- 色相ベースの虹色グラデーション
- OrbitControls によるカメラ操作
